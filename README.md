# News CTR Prediction (新聞點閱率預測模型)

本專案旨在建立一個可於「新聞發布前」預測點閱率（CTR, Click-Through Rate）的模型，並結合多模態特徵與檢索增強（Retrieval-Augmented Modeling）方法，提升熱門新聞的預測能力。

## 單篇新聞產製流程

選題與下標（依據時事／數據／熱搜） <- 此階段可使用模型，進行 CTR 預測評估標題潛力  
→ 確認可寫（符合編輯方向與新聞價值後進入撰稿）  
→ 撰寫稿件（內文撰寫、資料引用、圖片配置與事實查核）  
→ 提交審核（主管審核內容與標題並進行優化）  
→ 發布（上線並進入社群平台等流量分發管道）

## 專案目標

- 預測新聞標題的潛在點閱率（0~1）
- 輔助生成模型訓練
- 提供編輯下標階段決策輔助

---

## 資料來源

資料來自台灣新聞平台，包含：

- 標題（Title）
- 審核日期（Datetime）
- 總點閱數（PV）
- 熱搜關鍵字（Google Trends）


## 方法概述

### 1. 特徵工程

- **文本特徵**：BERT / Transformer（768 維）
- **圖片特徵**：CLIP 向量（512 維）
- **時間特徵**：weekday / weekend
- **分類器**：是否熱門的機率軟標籤（1 維）
- **結構特徵（可選）**：標題長度、標點數等

---

### 2. 模型架構

- 基礎模型：MLP Regression
- 輸入維度：1283 維（BERT + CLIP + 時間 + 分類器）
- 輸出：點閱率（0~1）

---
### 3. 檢索增強

建立歷史新聞資料庫：

- (title, embedding, PV)

預測流程：

1. 將輸入標題轉為向量
2. 檢索 Top-K 相似新聞
3. 根據相似度與 PV 計算 **RAG score**
4. 與模型輸出做加權平均 **主模型佔80%、RAG佔20%表現最佳**

## 實驗結果（Experiment Results）

本研究以多模態特徵（BERT + CLIP + 時間 + 分類器機率）建構新聞點閱率預測模型，並與傳統機器學習方法進行比較。實驗採用回歸指標（MAE、MSE、RMSE、R²）與分類指標（Accuracy、Recall、F1-score）進行評估。

結果顯示：

- 傳統模型（XGBoost、LightGBM）較適用於排序任務，對於實際點閱率數值預測能力有限  
- 多模態模型（MLP + BERT + CLIP）在整體預測表現上明顯優於傳統方法  
- 採用「分類再回歸」架構（加入熱門機率特徵）可進一步提升模型對高點閱新聞的辨識能力  

主模型最佳表現：

- Validation Loss：0.0526  
- MAE：0.1861  
- RMSE：0.2293  
- R²：0.4074  

分類模型結果：

- MLP + BERT + CLIP 在熱門新聞辨識（F1-score、Recall）上優於 XGBoost 與 LightGBM  

整體結果驗證，多模態特徵融合與階層式預測架構能有效提升新聞點閱率預測準確性，特別是在高點閱新聞的判別能力上具有顯著優勢。
