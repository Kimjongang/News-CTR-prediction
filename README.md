# News CTR Prediction (新聞點閱率預測模型)

本專案旨在建立一個可於「新聞發布前」預測點閱率（CTR, Click-Through Rate）的模型，並結合多模態特徵與檢索增強（Retrieval-Augmented Modeling）方法，提升熱門新聞的預測能力。

單篇新聞產製流程
-> 選題與下標（依據時事／數據／熱搜） <-此階段可使用模型，進行CTR預測評估標題潛力
　　　　-> 確認可寫（符合編輯方向與新聞價值後進入撰稿） 
    　　　　　　-> 撰寫稿件（內文撰寫、資料引用、圖片配置與事實查核） 
          　　　　　　　-> 提交審核（主管審核內容與標題並進行優化） 
                 　　　　　　　-> 發布（上線並進入社群平台等平台）
                        
---

## 專案目標

- 預測新聞標題的潛在點閱率（0~1）
- 輔助生成模型訓練
- 提供編輯下標階段決策輔助

---

## 資料來源

資料來自台灣新聞平台，包含：

- 標題（Title）
- 審核日期（Datetime）
- 總點閱數（PV）
- 熱搜關鍵字（Google Trends）


## 方法概述

### 1. 特徵工程

- **文本特徵**：BERT / Transformer（768 維）
- **圖片特徵**：CLIP 向量（512 維）
- **時間特徵**：weekday / weekend
- **分類器**：是否熱門的機率軟標籤（1 維）
- **結構特徵（可選）**：標題長度、標點數等

---

### 2. 模型架構

- 基礎模型：MLP Regression
- 輸入維度：1283 維（BERT + CLIP + 時間 + 分類器）
- 輸出：點閱率（0~1）

---
### 3. 檢索增強

建立歷史新聞資料庫：

- (title, embedding, PV)

預測流程：

1. 將輸入標題轉為向量
2. 檢索 Top-K 相似新聞
3. 根據相似度與 PV 計算 **RAG score**
4. 與模型輸出做加權平均 **主模型佔80%、RAG佔20%表現最佳**
